# 투포인터 알고리즘

## 정의
- 1차원 배열(혹은 리스트)에서 2개의 포인터를 통해서 원하는 값을 찾을때까지 탐색하는 알고리즘

## 동작 방식 & 구현
- 2개의 포인트 start&end 혹은 left&right 등 지정하기 나름이지만 left, right로 사용
- 제한 조건은 left <= right, 이고 처음 동작은 0, 0으로 시작한다.
